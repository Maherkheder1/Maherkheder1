Project Overview

AMANI is a service-based platform for a cosmetic and beauty center, offering nails, hair, makeup services, and professional training courses. The platform allows users to browse services, book appointments, purchase products, and manage their accounts.

The backend was built using Laravel, focusing on clean architecture, secure data handling, appointment management, and scalable API design. The system supports business operations such as scheduling, inventory, and order tracking.
